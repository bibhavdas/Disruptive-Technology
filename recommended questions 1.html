Part 2: Theoretical AI Questions

1) Difference between Supervised, Unsupervised, and Semi-Supervised AI Models
  • Supervised Learning: The algorithm is trained using well-labeled data. It maps input variables (X) to an output variable (Y). Example: Image classification (dog vs. cat).
  • Unsupervised Learning: The algorithm trains on unlabeled data, finding hidden patterns or intrinsic structures within the data. Example: Customer clustering based on purchasing habits.
  • Semi-Supervised Learning: A hybrid approach utilizing a small amount of labeled data alongside a large pool of unlabeled data, drastically reducing the cost of data labeling while improving accuracy.
2) Brief description about Reinforcement Learning

Reinforcement Learning (RL) is a behavior-based learning model where an agent learns to make decisions by taking actions in an environment to maximize cumulative rewards. It operates on a trial-and-error basis. It is widely used in robotics, gaming (e.g., AlphaGo), and autonomous vehicles.

4) Types of Data: Qualitative, Quantitative, Categorical, Continuous
  • Quantitative: Numerical data that can be measured objectively (e.g., Age, Height, Weight).
  • Qualitative: Non-numerical data that describes attributes or characteristics (e.g., Texture, Taste).
  • Categorical: A subset of qualitative data grouped into categories without a logical order (e.g., Blood Type: A, B, AB, O; Eye Color: Blue, Brown).
  • Continuous: A subset of quantitative data that can take any infinite value within a range (e.g., Temperature: 98.6°F, Exact distance: 5.432 miles).
5) What is NLP, Tokenization, Lemmatization

NLP (Natural Language Processing): A branch of AI enabling computers to comprehend, interpret, and manipulate human language.

Tokenization: Breaking text into smaller pieces called tokens (words, subwords, or sentences).
Ex: "AI is great" → ["AI", "is", "great"]

Lemmatization: Reducing a word to its base dictionary form (lemma) based on contextual meaning.
Ex: "Running", "Ran", "Runs" → "Run"

6) What is an LLM model?

LLM (Large Language Model): A type of foundational AI model trained on massive amounts of text data using Deep Learning (specifically Transformer architectures). They are capable of understanding context and generating human-like text, translating languages, and writing code. Examples include OpenAI's GPT-4, Google's Gemini, and Meta's LLaMA.

7) Best Fit, Overfit, and Underfit

These terms describe how well a machine learning model generalizes to new data.

Model Fitting Visualized

  • Underfit: The model is too simple (e.g., linear line for non-linear data). Performs poorly on both training and testing data.
  • Best Fit: The model captures the underlying trend without memorizing the noise. Generalizes well to unseen test data.
  • Overfit: The model is too complex. It memorizes the training data (including noise and outliers) perfectly but fails drastically on unseen test data.
8) Role of Data in Healthcare, Agriculture, and Education Industries
  • Healthcare: Data powers predictive diagnostics (forecasting patient readmissions), medical imaging (AI detecting tumors in X-Rays), drug discovery, and personalized medicine based on genetic profiles.
  • Agriculture: Through IoT sensors and satellite imagery, data enables precision farming. It tracks soil moisture, predicts weather patterns for optimal planting, automates crop disease detection, and maximizes yield.
  • Education: Data analytics fuels Adaptive Learning Systems that tailor curriculum to individual student pacing. It is also used to flag students at risk of dropping out and to automate administrative tasks like grading.
