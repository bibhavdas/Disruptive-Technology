Appropriate Fitting

In machine learning, the best fit model is one that optimally balances model complexity. It performs well on both training data and new unseen data. This balance is known as the Bias-Variance Trade-off. It ensures the model has strong generalizability and can make reliable predictions in real-world applications.

Under-fitting

Underfitting occurs when a model is too simple to capture the essential patterns in the data (High Bias). It performs poorly on both the training and test datasets.

Overfitting

Overfitting occurs when a model is too complex and learns the noise and random fluctuations in the training dataset (High Variance). It performs exceptionally well on training data but poorly on unseen data.

Supervised Learning

Supervised learning involves training a model on labeled data.

1. Regression

Regression in machine learning is a supervised learning technique used to predict a continuous numerical value based on one or more input or independent features.

In regression, the model learns the relationship between input features and output by fitting a mathematical function to the training data.

Examples: Predicting market salary, temperature, sales, etc.

2. Classification

Classification is a supervised learning technique used to predict a category or class label for given input data.

In classification, the model learns from labeled training data and assigns new data points to predefined classes.

Examples: Yes/No, True/False, Color grading, etc.
